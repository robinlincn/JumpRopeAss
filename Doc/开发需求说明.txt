湖南省学生跳绳协会微信小程序开发完整需求
一、项目定位
开发 “湖南省学生跳绳协会” 微信小程序，核心满足跳绳资讯展示、赛事全流程管理（创建 / 报名 / 缴费）、证书管理等需求，兼顾协会品牌展示与用户操作便捷性。
二、视觉与技术栈要求
1. 视觉风格
参考 Design 文件夹截图的栏目排版 / 内容框架，融合Ant Design系统风格（简洁、通透、年轻化），突出协会专属青春活力感，适配微信小程序视觉规范。
2. 技术栈
前端：先出设计稿，再根据确认好的设计稿形成H5页面，然后基于H5页面再做 uni-app 开发，仅适配微信小程序端；
后台管理端：采用ASP.NET开发，覆盖前端所有功能的配置与数据管理，后台管理界面风格参考Ant Design，代码和风格可以从https://github.com/ant-design/ant-design-pro获取；
数据库：MySQL（主机 / IP：175.6.7.5，端口：3306，数据库名：jumpropeass_student，用户名：root，密码：请在部署环境变量或 .env 中配置（禁止明文写入仓库））；需设计带中文注释的表结构，覆盖基础数据、赛事、报名、缴费、证书等核心维度。
三、核心功能需求
1. 基础数据管理
机构维度：分「培训机构」「中小学」两类，作为人员归属的基础；
人员维度：
裁判员 / 教练员 / 运动员 / 家长四类角色，其中教练员必须绑定培训机构 / 中小学；
运动员（核心为中小学学生）：必填绑定中小学，培训机构选填；人员信息变更（学校 / 培训机构）需留痕，支持追溯历史；
运动员「第一教练员」终身绑定，仅平台可修改；可自选第二、第三教练员（选填）；
家长：运动员注册时必填至少 1 位家长（姓名 + 手机号），支持绑定多位家长。
2. 赛事管理
赛事类型：
开放型：省协会等主办，全省 / 全国运动员可报名；
封闭型（预留功能）：学校 / 培训机构主办，仅旗下运动员可报名；
报名方式：支持运动员自主报名、家长代报名、教练员代报名（仅限旗下运动员）；
配套能力：支持赛事缴费、报名信息审核等核心流程。
3. 证书管理
证书类型：教练员证书、学员等级证书；
发证场景：首次发证、补证（两类场景支持分开定价）；
展示功能：用户可在小程序查看本人所有证书电子版。
4. 基础展示
支持跳绳相关新闻资讯的展示，适配整体页面风格与栏目排版要求。
